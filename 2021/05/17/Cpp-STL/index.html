大根堆与小根堆

  • 堆

    一种完全二叉树，可以用数组表示，i结点的父结点下标就为(i–1)/2。它的左右子结点下标分别为2 * i + 1和2 * i + 2。如第0个结点左右子结点下标分别为1和2。

  • 大根堆：父节点值必大于其子节点，根节点最大

    小根堆：父节点值必小于其子节点，根节点最小

Set 类

  • 在set中每个元素的值都唯一，而且系统能根据元素的值自动进行排序。set中元素的值不能直接被改变。set内部采用的是一种非常高效的平衡检索二叉树：红黑树，也称为RB树(Red-Black Tree)。RB树的统计性能要好于一般平衡二叉树。set中的元素都被排序好了，且唯一。

Map 类

map是STL(Standard Template Library)里重要容器之一。

它的特性总结来讲就是：所有元素都会根据元素的键值key自动排序（也可根据自定义的仿函数进行自定义排序），其中的每个元素都是<key, value>的键值对，map中不允许有键值相同的元素，

因此map中元素的键值key不能修改，但是可以通过key修改与其对应的value。如果一定要修改与value对应的键值key，可将已存在的key删除掉，然后重新插入。

  • map底层为红黑树查找大致为logN的时间复杂度,因为map需要排序；unordered_map底层是闭散列的哈希桶，查找为O(1)，性能更优。

Tips

  • upper_bound与lower_bound

    lower_bound( begin,end,num)：从数组的begin位置到end-1位置二分查找第一个大于或等于num的数字，找到返回该数字的地址，不存在则返回end。

    upper_bound( begin,end,num)：从数组的begin位置到end-1位置二分查找第一个大于num的数字，找到返回该数字的地址，不存在则返回end。

    通过返回的地址减去起始地址begin,得到找到数字在数组中的下标.
