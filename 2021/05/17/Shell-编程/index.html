Shell 编程

shell 编程

基础内容

  • echo 向控制台输出内容,echo $xxx即可输,y出变量
  • /etc/init.d/ 下为shell 教程，/etc/shells 中有其他shell库
  • #表示注释, 加减乘除等与C相同, 赋值不能含有空格，用VIM编辑，用txt文件编辑会有问题
  • 可通过如下方式运行shell脚本,一般用bash
  • !path 为解释器路径，shell编程都为.sh文件
位置变量 描述
$! 退出返回，为0则表示正常退出
$0 当前进程或脚本名称
$1-9 输入参数
$* 所有参数的整体
$@ 逐个读取所有参数
$! 后台运行的最后一个进程的PID

字符串

  • 字符串可以用单引号，也可以用双引号，也可以不用引号
  • 单引号不单独出现
  • 双引号里可以有变量，转义字符

数组与算式

  • 在shell 中括号为数组，数字间空格隔开

    array_name=(value0 value1 value2 value3)

  • 通过 ${数组名[下标]} 获取下标参数，@字符获得数组中全部元素，输出长度方法与字符串相同

  • 通过如下3种方式进行运算符操作

判断与其他运算符

  • ! , -a, -o 为 非，与(&&)，或(||)
  • 作用于字符串的运算符
运算符 说明 例子
= 字符串判等
!= 字符串判不等
-z 字符串长度为0返回true [ -z $a ]
-n 字符串长度不为0返回true [ -n $a ]
str 长度不为0返回true [ $a ]
  • 作用于数字的运算符
运算符 说明
-eq 两数相等返回true
-ne 两数不相等返回true
-gt 左>右返回true
-lt 左<右返回true
-ge 左>=右返回true
-le 左<=右返回true
  • 作用于文件的常用运算符
运算符 说明
-d 文件若是目录则返回true
-f 文件若是普通文件则返回true
-r/w/x 读/写/执行 的权限
-e 文件是否存在

if & switch

  • if 结构，注意 要和其他语句中间有空格，出现if必须跟着then
  • case 结构, 注意 ;;

for & while

  • for 结构,let中变量可不用$x 表示
  • while 结构

定义函数

  • read 输入信息，在函数后直接加要输入的实参即可,$?获取函数的return值
