vim编辑器

vim 编辑器

  • 在vim程序中设定的vim工作方式是临时的，如果要永久保存vim的工作方式需要编辑vim的配置文件:/etc/vimrc

  • 高亮模式进入vim

  • vi 或 vim 进入文件第5行

命令模式

  • 为默认模式，该模式无法输入

  • 命令 说明
    w 到下一个单词开头
    e 到当前单词结尾
    b 到前单词开头
    ge 到前单词结尾
    0或^ 到行头
    $ 到行尾
    f<> 向后搜索第一个<>内字符处
    F<> 向前搜索第一个<>内字符处
    yy 复制整行
    yw 复制一个单词
    p 粘贴至光标后(大写则为后)
    x或Delete 删除游标所在字符(X则为前一个)
    dd 删除整行
    5dd 删除5行
    nG 游标移动到第n行
    gg 游标移动到第一行
    G 游标移动到最后一行
    pgup/pgdn 上下翻屏
    5+上下左右 上下左右移动5行
    u 撤销
    ctrl+r 恢复撤销
    . 重复上次操作，除了ctrl+r

编辑模式

  • 输入 [ i, I, o, O, a, A, r, R]等字母之后进入编辑模式，可以做文字数据输入

  • a：在光标后开始插入

    A：在行尾开始插入

    i：从光标前开始插入

    I：从光标列的第一个非空白字元面前开始插入

    o：在光标所在列下新增一列并进入输入模式

    O：在光标所在列上方新增一列并进入输入模式

  • 该模式无法保存，需Esc退出至命令模式

末行模式

  • 命令模式输入 / 或 : 进入末行模式，esc退回命令模式
命令 说明
:syntax on/off 开启/关闭语法提示
:w 保存文件
:w path 存在path下
:wq 保存并退出，就算未修改文件也更改文件时间
:x 保存并退出，未更改文件就不改时间
:q! 不保存强制退出
:10 移动到第十行
:wq! 强制保存退出
! 使用外部命令，如!ls
:set nu 显示行号
:set nonu 取消显示行号
:u 撤销上一个命令
:X 设置密码，再次输入则解密
:hl 设置高亮，nohl取消高亮
:: something 查找something
  • :1,5s/:/@/g ##替换1-5行的：为@
  • :%s/:/@/g ##替换全文的：为@
  • :%s/要替换的字符/替换成的字符/g , 只替换一处
  • %s标示所有的行，g标示所有的列

多个文件处理

命令 说明
:bn 下一个文件
:bp 上一个文件
:open file2 打开另外的文件
:files 查看各文件，%a为当前活动，#为上一个打开的

vim的配置

  • 临时配置：命令行中的修改

  • 个人配置：在 user/vimrc中修改

  • 全局配置：etc/vimrc中

  • 临时>个人>全局

  • 若vim未保存退出，可删除同目录下的 .passwd.swp避免报错
